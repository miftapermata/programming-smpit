Hari ini kita belajar membuat halaman web dengan HTML dan menulis teks dalam bentuk paragraf.

Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. Bahasa ini digunakan untuk membuat struktur halaman web.

Kesimpulan

Dengan HTML, kita bisa menampilkan teks, gambar, link, dan elemen lainnya di web.
